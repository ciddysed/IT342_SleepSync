# IT342_SleepSync

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/IT342_SleepSync.git
   cd IT342_SleepSync
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment variables:
   - Create a `.env` file in the root directory.
   - Add the required environment variables as specified in `.env.example`.

4. Start the application:
   ```bash
   npm start
   ```

## Dependencies

- Node.js (v16 or higher)
- npm (v8 or higher)
- Additional dependencies are listed in `package.json`.

## Usage Guide

1. Run the application:
   ```bash
   npm start
   ```

2. Access the application in your browser at:
   ```
   http://localhost:3000
   ```

3. Follow the on-screen instructions to use the SleepSync features.

4. For development mode with live reloading:
   ```bash
   npm run dev
   ```

5. Run tests:
   ```bash
   npm test
   ```
